# Recipe App

Final task for REACT23K React Basics course.

## Features

User can:
- add new recipes
- view added recipes and search recipes by their name or country origin
- view recipe details

## Technologies used

Built with:

- HTML
- CSS
- JavaScript
- React
- React Router
- JSON server
- Axios

## Setup and Usage
- Clone repository
```
git clone https://github.com/julilan/recipe_app.git
```
- Install packages
```
npm install
```
- Run the app in development mode
```
npm start
```
- Start JSON server
```
npm run server
```
- Find app at http://localhost:3000/ (default port)
